# LeiloesTDSat

## Descrição do Projeto
Este projeto consiste em um sistema informatizado para uma casa de leilões, desenvolvido para gerenciar rotinas de venda e controle de produtos. Ele faz parte das atividades práticas de segurança e qualidade de software.

## Tecnologias Utilizadas
* **Java**: Linguagem principal do sistema.
* **MySQL**: Sistema de gerenciamento de banco de dados (SGBD) utilizando o banco `uc11`.
* **Git & GitHub**: Ferramentas utilizadas para o versionamento do código.
